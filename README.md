## Install the dependencies
```bash
yarn
# or
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn dev
# or
npm run dev.
```


### Lint the files (optional)
```bash
yarn lint
# or
npm run lint
```


### Format the files (optional)
```bash
yarn format
# or
npm run format
```



### Build the app for production
```bash
quasar build
```
